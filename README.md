**Ndako243** 🏠

**Ndako243** est une application web de recherche immobilière dédiée au marché de la République Démocratique du Congo. Ce projet est développé dans le cadre de mes études en développement logiciel à **Ensign College / BYU-Idaho**.

## 🚀 Objectif
L'objectif est de faciliter la mise en relation entre bailleurs et locataires grâce à une interface moderne, intuitive et rapide.

## 🛠️ Technologies utilisées
* **HTML5** : Structure sémantique des pages.
* **CSS3** : Design responsive avec l'utilisation de **Flexbox** et une palette de couleurs professionnelle (Dark Blue).
* **Git & GitHub** : Gestion de version et déploiement.

## 📈 Fonctionnalités actuelles
* Barre de recherche multicritères (Ville, Commune, Quartier, numéro parcelle).
* Affichage dynamique des maisons disponibles sous forme de cartes (Cards).
* Interface adaptée aux écrans d'ordinateurs et bientôt aux mobiles.
